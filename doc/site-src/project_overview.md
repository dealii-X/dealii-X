# Overview

dealii-X is a modular collection of libraries and applications for developing
high-performance organ simulators built on top of `deal.II`. This page is a
short summary for the site: it is not full technical documentation but an
introduction to the key concepts presented in the technical repositories section.

- Purpose: enable development and sharing of reusable components for
  organ simulations (brain, heart, lungs, liver, mechanobiology).
- Audience: researchers and developers integrating `deal.II` with
  solvers, preconditioners, and multi-physics coupling tools.

See the technical scope at `technical_scope.md`.
